# Pocket QSO

Minimal ham radio QSO log for Delta Chat (webxdc). Add contacts, edit/delete entries, and import/export CSV.

## Build

```sh
zip -r pocket-qso.xdc manifest.toml index.html main.js style.css
```

## Notes

- `webxdc.js` is provided by Delta Chat and must not be included in the `.xdc` archive.
